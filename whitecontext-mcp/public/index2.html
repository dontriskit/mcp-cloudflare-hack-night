About

WhiteContext MCP Server provides automated company intelligence analysis for AI assistants. Submit any company URL and receive comprehensive business intelligence including:

  • → Business models and revenue strategies
  • → Pricing structures and customer segments
  • → Technology stacks and integrations
  • → Competitive advantages and market positioning
  • → Customer case studies and testimonials
  • → Compliance certifications and security

Available Tools

📥 Retrieval Tools

get_job_results
Retrieval
Fetch complete analysis results for a job. Automatically cached for performance. Supports JSON and CSV formats.
get_batch_results
Retrieval
Retrieve results for multiple companies analyzed in a batch. Perfect for comparative analysis.
check_job_status
Retrieval
Quick status check without polling. Returns processing stage and completion status.
download_results
Retrieval
Export results in JSON or CSV format. Useful for spreadsheets and data analysis.
list_jobs
Retrieval
View recent job history with status and timestamps. Find job IDs easily.

📤 Submission Tools

submit_job
Submission
Submit a single URL for analysis. Returns job_id immediately for async tracking.
submit_bulk_job
Submission
Submit up to 100 URLs for batch analysis. Ideal for competitive intelligence research.

⚡ Auto-Polling Tools

analyze_url_wait
Auto-Poll
Submit URL and automatically wait for completion (up to 5 minutes). Returns full results.
analyze_bulk_wait
Auto-Poll
Submit multiple URLs and wait for batch completion (up to 10 minutes). All results returned.

🔧 Management Tools

configure_webhook
Config
Configure webhook URL to receive notifications when jobs complete.
clear_cache
Config
Clear cached results for a specific URL or all cached results.

Quick Start

Connect to this MCP server using Claude Code or any MCP-compatible client:

# Install the MCP server claude mcp add --transport http whitecontext-local \ https://whitecontext-mcp.maksym.workers.dev/mcp # Or add to your .mcp.json { "mcpServers": { "whitecontext-local": { "type": "http", "url": "https://whitecontext-mcp.maksym.workers.dev/mcp" } } }

Example usage in Claude Code:

// Analyze a single company > Analyze https://stripe.com and give me their pricing model // Batch analysis > Compare these 5 companies: [URLs] // Get structured data > List all my recent analysis jobs
