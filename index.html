What is HONC?

🚀

Hono Framework

Ultra-lightweight (14KB) TypeScript API framework built for Web Standards environments. Fast, modern, and works everywhere: Cloudflare, Deno, Bun, Vercel, Node.

🗄️

Drizzle ORM

Type-safe query builder for relational databases. Seamless integration with Cloudflare D1 (SQLite), Neon, Supabase, and vector stores for AI.

☁️

Cloudflare Edge

Deploy to 330+ cities globally with Workers, D1, KV, Queues, and R2. Zero cold starts, <50ms latency for 95% of the world.
